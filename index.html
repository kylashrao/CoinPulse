Why we say "refreshed every minute," not "live"

Many crypto price sites use the word "live" loosely. In practice, free-tier price data — including the data this site uses — is typically cached for one to five minutes before it updates on the data provider's end. Polling more frequently than that wouldn't give you fresher numbers; it would just waste requests. CoinPulse checks for new data every 60 seconds and always shows you exactly when the price was last updated, so you can judge its freshness yourself instead of taking "live" on faith.

Where this data comes from

Prices on CoinPulse are pulled directly from CoinGecko's public market data API, which aggregates prices across a large number of exchanges worldwide. We don't run our own price feed or hold any cryptocurrency ourselves — we simply display what CoinGecko's API reports, refreshed on the schedule described above.

A note on currencies

Switching the currency toggle changes the displayed price to US Dollars, British Pounds, Canadian Dollars, or Australian Dollars, each converted using the exchange rate CoinGecko's API provides at the time of the request.
